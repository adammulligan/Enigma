\abstract{
	\addtocontents{toc}{\vspace{1em}}  

	A vital part of modern-day communication is the ability to converse with confidence that you are not being eavesdropped on -- a confidence that is ensured through the use of digital cryptography. Contemporary cryptography provides end-to-end solutions for this, including key-agreement, encryption, and certification, most of which are based on fundamental number theory problems.

	This report discusses the process of understanding, creating and using these solutions in a fully-fledged application and any implications and issues that arise.

	Following this guide will provide with all the information necessary to understand and create a secure instant messaging system with the option of implementing multiple cryptographic algorithms for all information security requirements.
}
